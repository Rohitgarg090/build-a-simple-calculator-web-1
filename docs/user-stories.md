# User Stories

## Calculator Operations
- **US-001**: As a user, I want to click number buttons to input digits so that I can form a number to calculate with.
- **US-002**: As a user, I want to click an operator button (+, -, ×, ÷) so that I can define the arithmetic operation to perform.
- **US-003**: As a user, I want to press the `=` button so that I can see the result of my expression.
- **US-004**: As a user, I want to press `AC` so that I can reset the calculator to its initial state.
- **US-005**: As a user, I want to press `CE` so that I can clear only the last entered digit or value without resetting the full expression.
- **US-006**: As a user, I want to input a decimal point so that I can work with non-integer numbers.
- **US-007**: As a user, I want to see an error message when I divide by zero so that I understand the operation is invalid.
- **US-008**: As a user, I want to use my keyboard to type numbers and operators so that I can calculate faster without using the mouse.
- **US-009**: As a user, I want to chain multiple operations so that I can perform multi-step calculations without resetting.

## History
- **US-010**: As a user, I want each completed calculation to be automatically saved so that I can refer back to previous results.
- **US-011**: As a user, I want to navigate to a history page so that I can view all my past calculations.
- **US-012**: As a user, I want to see the expression and result for each history entry so that I understand what was calculated.
- **US-013**: As a user, I want to see the timestamp of each calculation so that I know when it was performed.
- **US-014**: As a user, I want to clear all history so that I can remove records I no longer need.
- **US-015**: As a user, I want history to be paginated so that the page does not become slow with many records.

## UI/UX
- **US-016**: As a user, I want the calculator to be usable on my phone so that I can calculate on the go.
- **US-017**: As a user, I want visual feedback when I press a button so that I know my input was registered.
