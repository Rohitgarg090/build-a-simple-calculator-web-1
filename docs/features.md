# Features

## Core Calculator
- **Basic Arithmetic**: Addition, subtraction, multiplication, division
- **Display Panel**: Shows current input and running expression
- **Clear & Reset**: AC (All Clear) and CE (Clear Entry) buttons
- **Decimal Support**: Floating point number input
- **Equals Evaluation**: Compute and display result on `=` press
- **Chained Operations**: Continue calculating from previous result
- **Keyboard Support**: Accept input from physical keyboard

## History
- **Save Calculations**: Automatically save each completed calculation to Supabase
- **View History**: Paginated list of past calculations with expression and result
- **Clear History**: Delete all saved history records
- **Timestamps**: Show when each calculation was performed

## UI/UX
- **Responsive Design**: Works on mobile, tablet, and desktop
- **Tailwind Styling**: Clean dark-themed calculator layout
- **Button Feedback**: Hover and active states on all keys
- **Error Handling**: Display 'Error' on invalid operations (e.g., divide by zero)
